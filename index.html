simpleButton

A simple CSS3 button with 3D effect

Basic usage:

The "buttons" can be created by adding class="button" to any appropriate <a>, <button>, or <input> element.

<button class="button">Próximo passo</button>

Add a class has-icon to create a icon to the left - you need to add a <span> element with a class="icon" class and an available class of icon.

<a href="#" class="button has-icon"><span class="icon next"></span>Próximo passo</a>

Próximo passo

More examples:

<a href="#" class="button has-icon disabled"><span class="icon next"></span>Próximo passo</a>
